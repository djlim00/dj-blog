## 헤더
___

- ':'으로 구분되는 key-value형태
- 3가지의 헤더인 인반헤더, 요청헤더, 응답헤더가 자동으로 생김
- 서버에서 설정하는 헤더 -> 응답헤더
- 클라이언트에서 설정하는 헤더 -> 요청헤더

### 일반헤더
![[Pasted image 20250217130602.png]]
- 요청한 URL
- 요청 메서드
- 해당 자원을 요청할 떄 해당자원의 출처를 나타내는 URL의 노출여부의 보안정도의 Referrer Policy정도가 들어감


### 요청헤더
![[Pasted image 20250217130647.png]]
- 클라이언트가 서버에 요청할 때 클라이언트가 설정하는 또는 자동으로 설정되는 헤더
- 메서드, 클라이언트IS, 브라우저 정보,,,etc

### 응답헤더
![[Pasted image 20250217130830.png]]
- 서버가 클라이언트에게 응답할 때 설정하는 또는 자동으로 설정되는 헤더
- 자로 서버의 소프트웨어 정보
	- ex) nginx를 프록시 서버로 두었다는 정보
	- <span style="background:rgba(205, 244, 105, 0.55)">일반적으로 해커가 서버의 소프트웨어를 알기 어렵게 하기 위해 숨김</span>
	- server : NWS라고만 써져있고, 서버가 어떤 소프트웨어를 쓰는지는 안나옴!
