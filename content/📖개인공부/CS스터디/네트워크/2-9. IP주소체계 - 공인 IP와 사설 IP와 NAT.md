# NAT이란?
____
- <span style="background:rgba(205, 244, 105, 0.55)">NAT(Network Address Translation)은 패킷이 트래픽 라우팅 장치를 통해 전송되는 동안 패킷의 IP주소를 변경, IP주소를 다른 IP주소로 매핑하는 방법</span>

#### 외부와 통신할 때는 하나의 공인 IP를 사용하지만, 내부에서는 각각의 IP를 가지고 있음
