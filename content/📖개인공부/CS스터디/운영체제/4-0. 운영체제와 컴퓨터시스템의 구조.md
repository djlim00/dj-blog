### GUI
Graphical User interface -> 그래픽을 사용하여 컴퓨터와 상호작용하는 인터페이스
ex) Windows, macOs 등의 현재 OS

Character User interface -> 키보드만을 사용하여 문자를 기반으로 컴퓨터와 상호작용하는 인터페이스

# 운영체제의 역할
___
- CPU 스케쥴링 및 프로세스 상태 관리 
- 메모리 관리
- 디스크 파일 관리 
- I/O 디바이스 관리 - 키보드, 마우스, 모니터

보통 OS는 커널이랑 거의 같은 의미로 쓰임
프로그램 실행 -> 램이 올라감 -> 프로세스가 됨 -> 프로세스를 CPU가 실행함
많은 프로그램을 실행 -> 램이 꽉차게 된다 -> 메모리가 부족하면 Swapping이 발생 -> 우선순위를 기반으로 실행 -> Interrupt발생 -> ...etc
=> <mark style="background: #FFF3A3A6;">이 모든 것들을 OS가 하는 것</mark>

# CPU
___
- 산술논리장치, 제어장치, 레지스터로 구성된 장치
- <mark style="background: #FFF3A3A6;">인터럽트</mark>에 의해 메모리에 존재하는 명령어를 해석해서 실행하는 일꾼

#### 1. 산술논리장치 ALU
![[Pasted image 20250317131632.png]]
- 사칙연산과 같은 산술연산과, && ||같은 노리연산을 하는 회로장치

#### 2. 제어장치(Control Unit, CU)
- 프로세스의 조작을 지시
- 명령어들을 일곡 해석하며 데이터 처리를 위한 순서를 결정

#### 3. 레지스터
- CPU안에 있는 매우 빠른 임시기억장치
