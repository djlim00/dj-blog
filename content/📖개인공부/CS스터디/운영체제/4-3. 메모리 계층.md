레지스터, 캐시, 주기억장치, 보조기억장치

1. 레지스터 : CPU 내의 작은 메모리, 휘발성, 속도 가장 빠름, 기억용량 작음
2. 캐시 : CPU내의 L1, L2 캐시, 휘발성, 속도 빠름, 기억용량 작음
3. **주기억장치** : **RAM**을 의미. 휘발성, 속도 보통, 기억용량 보통
4. 보조기억장치 : HDD, SDD를 의미. 비휘발성, 속도 낮음, 기억용량 많음

# 왜 계층이 존재할까?
____
## 1. 더 빠른 접근과 처리속도 증가
- 우리는 보통 많이 쓰는 것을 다시 많이 쓴다.
- 특정 데이터에 더 자주 접근하게 되므로, 캐시 메모리에 해당 데이터가 있다면 더 빠르게 해당 데이터에 접근 가능
- 즉 처리 속도 증가

### 2. 비용의 효율성
- 캐시 메모리는 비싸고 램 등 아래로 내려갈 수록 저렴
- 비용을 효율적으로 사용할 수 있음

### 3. 자원의 효율적 사용
- 자주 접근하는 데이터는 빠른 메모리 / 덜 접근하는 데이터는 느린 메모리에 저장
	- 자원을 효율적으로 사용
- 그렇지 않으면 거의 접근하지 않는 데이터에 빠른 메모리를 할당해서 자원을 낭비
