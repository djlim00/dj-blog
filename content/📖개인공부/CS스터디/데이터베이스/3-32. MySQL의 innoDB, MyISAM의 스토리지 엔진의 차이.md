
- <span style="background:rgba(3, 135, 102, 0.2)">스토리지 엔진은 데이터베이스 시스템에서 데이터의 저장, 검색, 트랜잭션 관리, 동시성 제어 , 캐싱 등 핵심을 담당하는 필수 구성 요소임</span>
	- InnoDB : MySQL 8.0의 기본 스토리지 엔진, MyISAM 으로도 변경이 가능하고 차이가 있음

-> innoDB는 행 레벨로 락을 걸어서, 다른 행이면 서로 간섭을 안함
-> MyISAM은 테이블을 락을 걸어버림(같은 테이블에 접근을 할 수가 없음)
-> 외래 키는 innoDB는 지원하지만, MyISAM은 지원하지 않는다.
