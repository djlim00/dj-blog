	ENUM과 SET 모두 문자열을 열거한 타입이다.
<mark style="background: #FFF3A3A6;">지정된 문자열 이외의 값이 들어가게 되면 에러가 발생된다.</mark>

# ENUM
___
- 열에 할당할 수 있는 값의 리스트를 정의함 - <mark style="background: #FFF3A3A6;">이 리스트 중 하나만 선택하는 단일선택만이 가능</mark>
- ex) ENUM('red', 'blue', 'green')
	- 세 개중 하나만 저장할 수 있음 , 이외의 값을 저장하면 MySQL은 에러를 발생시킴
	- <font color="#d83931">내부적으로는 red: 1, blue : 2, green : 3 이렇게 숫자로 저장</font>

```sql
use csnote;

CREATE TABLE shirts (
21id INT NOT NULL AUTO_INCREMENT,
color ENUM('red', 'green', 'blue') NOT NULL,
PRIMARY KEY (id)
);

INSERT INTO shirts (color) VALUES ('red'), ('blue'), ('green');

SELECT * FROM shirts;
```

# SET
___
- ENUM과 비슷하지만
- <mark style="background: #FFF3A3A6;">하나의 열에 여러 값을 저장할 수 있음</mark>
- <mark style="background: #FFF3A3A6;">비트단위 연산이 가능</mark>
- <mark style="background: #FFF3A3A6;">한번에 여러개의 조합으로 선택이 가능</mark>
- <mark style="background: #FFF3A3A6;">64개의 요소를 넣을 수 있음</mark>

```sql
use csnote;

CREATE TABLE socks (
id INT NOT NULL AUTO_INCREMENT,
colors SET('red', 'green', 'blue') NOT NULL,
PRIMARY KEY (id)
22);

INSERT INTO socks (colors) VALUES ('red'), ('red,green'),

('green,blue'), ('red,green,blue');

SELECT * FROM socks;
```
