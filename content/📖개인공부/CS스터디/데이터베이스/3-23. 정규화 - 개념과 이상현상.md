## 이상현상(anomaly)
____
- 데이터베이스 설계를 잘못하게 되었을 때, <span style="background:rgba(3, 135, 102, 0.2)">불필요한 데이터 중복</span>이 발생하여 
- 데이터의 삽입, 갱신, 삭제 연산을 수행할 때 부작용이 발생(조회는 X)

### ex) 무신사
![[Pasted image 20250306183812.png]]
- 뭔가 하나의 row에 너무 많은 정보가 있음

### 이상현상 - 삽입이상

- 불필요한 데이터도 같이 삽입을 해야함
- <mark style="background: #FFF3A3A6;">이벤트 참여를 안하는 경우에는 NULL값이 들어가게됨</mark>
![[Pasted image 20250307171034.png]]

### 이상현상 - 갱신이상

- <mark style="background: #FFF3A3A6;">중복 레코드가 있고, 이 중 일부만 변경해 데이터 자체가 불일치 되는 문제가 발생</mark>
- 주홍철의 등급이 플레티넘으로 승급했음에도 불구하고 일부만 수정됨
![[Pasted image 20250307171136.png]]
- 딱 하나만 플레티넘으로 수정되고, 다른 데이터는 여전히 골드임..

### 이상현상 - 삭제이상

- 필요한 데이터까지 함께 삭제되는 문제
- 만약 이벤트 참여를 취소한다면?? -> 고객 아이디, 이름까지 삭제 해야함
