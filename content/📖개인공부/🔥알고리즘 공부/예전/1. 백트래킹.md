### 백트래킹과 DFS

- 기본적으로 깊이우선탐색은(dfs)는 완전탐색 알고리즘이므로, 모든 경우의 수를 전부 돈다.
- 백트래킹은 쉽게 생각하면 dfs를 하는 도중 **현재의 경로가 해가 되지 않을 것 같으면 더 이상 진행하지 않고 돌아오는 것**이다.
- 가지치기를 얼마나 잘 하냐에 따라서 효율성이 결정됨!
- 진행중 답이 되지 않는 조건문을 이용해서 부모노드로 돌아가게 해야함

# 백준15686번 - 치킨배달

https://www.acmicpc.net/problem/15686

```java title:"소스코드"
import java.io.*;  
import java.util.ArrayList;  
import java.util.StringTokenizer;  
  
public class BOJ15686_치킨배달_임동준 {  
    static BufferedReader br = new BufferedReader(new InputStreamReader(System.in));  
    static BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));  
    static StringTokenizer st;  
    static int N, M;  
    static ArrayList<Position> houses;  
    static ArrayList<Position> kfcs;  
    static boolean[] visited;  
    static int minCityKFCDist = Integer.MAX_VALUE;  
  
    // N <= 50이므로 N^2으로 해도 될 것 같은디  
    public static void main(String[] args) throws IOException {  
        st = new StringTokenizer(br.readLine());  
        N = Integer.parseInt(st.nextToken()); // 도시 크기  
        M = Integer.parseInt(st.nextToken()); // 남은 치킨집 수  
        houses = new ArrayList<>();  
        kfcs = new ArrayList<>();  
        for (int r = 0; r < N; r++) {  
            st = new StringTokenizer(br.readLine());  
            for (int c = 0; c < N; c++) {  
                int num = Integer.parseInt(st.nextToken());  
                if (num == 1) houses.add(new Position(r, c));  
                else if (num == 2) kfcs.add(new Position(r, c));  
            }  
        }  
        visited = new boolean[kfcs.size()];  
  
        backtracking(0, 0);  
  
        bw.write(String.valueOf(minCityKFCDist));  
        bw.flush();  
    }  
  
    private static void backtracking(int cnt, int idx) {  
        if (cnt == M) {  
            int total = 0;  
            for (int i = 0; i < houses.size(); i++) {  
                int sum = Integer.MAX_VALUE;  
                for (int j = 0; j < kfcs.size(); j++) {  
                    if (visited[j]) {  
                        int dist = getDistance(houses.get(i), kfcs.get(j));  
                        sum = Math.min(sum, dist);  
                    }  
                }  
                total += sum;  
            }  
            minCityKFCDist = Math.min(total, minCityKFCDist);  
            return;  
        }  
        for (int i = idx; i < kfcs.size(); i++) {  
            if (!visited[i]) {  
                visited[i] = true;  
                backtracking(cnt + 1, i + 1);  
                visited[i] = false;  
            }  
        }  
    }  
  
    private static int getDistance(Position p1, Position p2) {  
        return Math.abs(p1.r - p2.r) + Math.abs(p1.c - p2.c);  
    }  
  
    private static class Position {  
        int r, c;  
  
        public Position(int r, int c) {  
            this.r = r;  
            this.c = c;  
        }  
    }  
}
```

- 백트래킹 함수안에 저렇게 전부 무식하게 계산을 할 생각을 못했는데, 결국은 치킨집들의 조합을 구하고, 거기서 최소의 거리를 구하고, 계속해서 최솟값을 갱신해 나가는 문제이다.
- 백트래킹이라고 볼 수 있는 것은, visited를 true로 하고 확인한 다음에, 재귀가 끝나면 다시 false로 만들어서 조합을 구한다.
