### 순열(Permutation) : 순서가 있는 경우의 수를 뽑는 것
___
- 우선 선택됐음을 알 수 있는 visited[ ] 배열이 필요함
1. 스택에 값을 넣고 true로 바꾼다
2. 1~N을 두고 반복문을 돌리면서 true로 바꾼다.
3. 반복문 사이에 재귀로 반복한다.

```java title:"백준10974 모든순열", hl:29-48
package A0210;  
  
import java.io.*;  
import java.util.Stack;  
  
public class 모든순열_순열 {  
  
    static BufferedReader br = new BufferedReader(new InputStreamReader(System.in));  
    static BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));  
  
    static int N;  
    static int[] arr;  
    static boolean[] visited;  
    static Stack<Integer> stack;  
  
    public static void main(String[] args) throws IOException {  
        N = Integer.parseInt(br.readLine());  
        arr = new int[N];  
        for (int i = 0; i < N; i++) {  
            arr[i] = i + 1;  
        }  
  
        visited = new boolean[N];  
        stack = new Stack<>();  
  
        permutation();  
    }  
  
    private static void permutation() throws IOException {  
  
        if (stack.size() == N) {  
            for (int i : stack) {  
                System.out.print(i + " ");  
            }  
            System.out.println();  
            return;  
        }  
        for (int i = 0; i < N; i++) {  
            if (!visited[i]) {  
                visited[i] = true;  
                stack.push(arr[i]);  
                permutation();  
                stack.pop();  
                visited[i] = false;  
            }  
        }  
    }  
}  
```

- 오케이 이건 재귀잖아? 그리고 반목문 안에 들어있어.
- 1을 당연히 먼저 선택해서 true, 그리고 재귀 들어가기(1) -> 1은 패스 2 스택에 넣음 -> 다시 재귀들어감(2) -> 1,2패스 3 들어감 -> 다시 재귀 들어감(3) -> size()== N 이니까 출력 -> return -> 3 pop, false -> (3)은 3까지 다 봐서 반복문 끝남 -> (2)로 돌아감 -> 2 pop, false -> 반목문 i=3으로 감 -> 3넣고 재귀 들어감 -> 2는 true -> 1,3,2 들어감 -> 다시 재귀 탈출 -> (2) 끝남 -> (1) 돌아감 -> 1 pop, false -> (1)의 i가 2가 됨 -> 2로 재귀 시작 -> .......
- 이거 그냥 인자 두개 받아서 stack.size()== r로 하면 nCr 구현완료
