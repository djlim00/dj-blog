### 1. 네츄럴 템플릿 - 순수 HTML을 유지하는 뷰템플릿

- **최대한 순수 HTML을 유지하는 특징**
- 웹 브라우저에서 직접 열어도 내용을 확인할 수 있고, 서버를 통해 뷰 템플릿을 거치면 동적으로 변경된 결과를 확인할 수 있음.
	- <mark style="background: #FFF3A3A6;">JSP 코드는 자바 코드가 HTML 사이에 섞여있어서, 웹브라우저(크롬)에서 정상적인 HTML을 확인할 수 없음.</mark>
- 타임리프는 파일을 웹 브라우저에서 바로 열어도 정상적인 HTML결과가 나온다(물론 동적으로 들어가는 값은 렌더링 안됨) 
___
### 2.  스프링 통합 지원

- 스프링과 자연스럽게 통합되고, 스프링 기능을 잘 쓸 수 있음.
- 
 ```java title:"타임리프 기본 표현식"
• 간단한 표현:
	
	◦ 변수 표현식: ${...}
	
	◦ 선택 변수 표현식: *{...}
	
	◦ 메시지 표현식: #{...}
	
	◦ 링크 URL 표현식: @{...}
	
	◦ 조각 표현식: ~{...}

• 리터럴

	◦ 텍스트: 'one text', 'Another one!',…
	
	◦ 숫자: 0, 34, 3.0, 12.3,…
	
	◦ 불린: true, false
	
	◦ 널: null
	
	◦ 리터럴 토큰: one, sometext, main,…

• 문자 연산:

	◦ 문자 합치기: +
	
	◦ 리터럴 대체: |The name is ${name}|

• 산술 연산:
	
	◦ Binary operators: +, -, *, /, %
	
	◦ Minus sign (unary operator): -


• 불린 연산:
	
	◦ Binary operators: and, or
	
	◦ Boolean negation (unary operator): !, not
	
	• 비교와 동등:
	
	◦ 비교: >, <, >=, <= (gt, lt, ge, le)
	
	◦ 동등 연산: ==, != (eq, ne)

• 조건 연산:
	
	◦ If-then: (if) ? (then)
	
	◦ If-then-else: (if) ? (then) : (else)
	
	◦ Default: (value) ?: (defaultvalue)

• 특별한 토큰:
	
	◦ No-Operation: _

```
