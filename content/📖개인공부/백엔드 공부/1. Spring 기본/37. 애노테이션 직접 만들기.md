- @Quilifier("이무동준") : 이렇게 쓰면 오타가 발생해도 문자열이기 때문에 컴파일 시 오타가 발생해도 알 수가 없다.
-> <font color="#ffff00">Annotation을 만들면 이러한 문제를 해결할 수 있다.</font>


```java title:"어노테이션 만드는 코드 / 어노테이션으로 선언", hl:29
package hello.example.core.annotation;  
  
  
import org.springframework.beans.factory.annotation.Qualifier;  
  
import java.lang.annotation.*;  
  
@Target({ElementType.FIELD, ElementType.METHOD, ElementType.PARAMETER, ElementType.TYPE, ElementType.ANNOTATION_TYPE})  
@Retention(RetentionPolicy.RUNTIME)  
@Inherited  
@Documented  
@Qualifier("mainDiscountPolicy")  
public @interface MainDiscountPolicy {  
}

============================ < 적용한 코드 > ================================

package hello.example.core.discount;  
  
import hello.example.core.annotation.MainDiscountPolicy;  
import hello.example.core.member.Grade;  
import hello.example.core.member.Member;  
import org.springframework.beans.factory.annotation.Qualifier;  
import org.springframework.context.annotation.Primary;  
import org.springframework.stereotype.Component;  
  
@Component  
//@Primary  
@MainDiscountPolicy  
public class RateDiscountPolicy implements DiscountPolicy {  
  
    private int discountPercent = 10;  
  
    @Override  
    public int discount(Member member, int price) {  
        if (member.getGrade() == Grade.VIP) {  
            return price * discountPercent / 100;  
        } else {  
            return 0;  
        }  
    }  
}
```


```java title:"어노테이션으로 Qualifier를 쓰기", hl:9

@Component  
public class OrderServiceImpl implements OrderService {  
  
    private final MemberRepository memberRepository;  
    private final DiscountPolicy discountPolicy;  
  
    @Autowired // - 생성자가 하나면 안써도 괜찮음  
    public OrderServiceImpl(MemberRepository memberRepository, @MainDiscountPolicy DiscountPolicy discountPolicy) {  
        this.memberRepository = memberRepository;  
        this.discountPolicy = discountPolicy;  
    }  
  
    @Override  
    public Order createOrder(Long memberId, String itemName, int itemPrice) {  
        Member member = memberRepository.findById(memberId);  
        int discountPrice = discountPolicy.discount(member, itemPrice);  
  
        return new Order(memberId, itemName, itemPrice, discountPrice);  
    }  
  
    public MemberRepository getMemberRepository() {  
        return memberRepository;  
    }  
}
```


- 사실상 그냥 @Qualifier를 사용하는 방식이랑 동일 -> 문자열을 기피해서 디버깅이 쉽게 만드는 것
- Qualifier 뿐만 아니라 다른 어노테이션과 엮어서 사용할 수 있다.
