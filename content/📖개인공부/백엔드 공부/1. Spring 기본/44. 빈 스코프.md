- 스프링 빈은 <font color="#ffff00">싱글톤 스코프</font>로 존재하기 때문에 스프링 컨테이너가 시작과 함께 생성되어, 종료와 함께 소멸된다.


### 스프링이 지원하는 다양한 스코프
1. <font color="#ffff00">싱글톤(default scope)</font>
   : 스프링 컨테이너의 시작과 종료까지 유지되는 가장 넓은 범위의 스코프
2. <font color="#ffff00">프로토타입</font>
   : 스프링 컨테이너는<u> 프로토타입 빈의 생성과 의존관계 주입까지만 관여</u>하고, 더는 관리하지 않는 ==매우 짧은 범위의 스코프==
   초기화 메서드까지 하고 클라이언트에 반환을 해준 다음은 스프링 컨테이너에서 관리하지 않는다.
**"<font color="#ffff00">웹 관련 스코프</font>"**
3. request : 웹 요청이 들어오고 나갈때 까지 유지되는 스코프
4. session : 웹 세션이 생성되고 종료될 때 까지 유지되는 스코프
5. application : 웹의 서블릿 컨텍스와 같은 범위로 유지되는 스코프


```java title:"컴포넌트 스캔 자동등록"
@Scope("protoype")
@Component
public class HelloBean{}
```

```java title:"수동 등록"
@Scope("prototype)
@Bean
PrototypeBean HelloBean(){
	return new HelloBean();
}
```
