![[Pasted image 20241231122938.png]]
- 전송 제어, 순서, 검증 정보 -> <font color="#ffff00">순서 제어 문제 해결</font>

![[Pasted image 20241231123059.png]]

- **연결 지향** - <font color="#92d050">연결이 됐는지 확인</font> 하고 보냄

- **순서 보장**


## 1.1 TCP 3 way handshake

![[Pasted image 20241231123157.png]]
> 1. SYN(싱크) 메세지를 서버로 보냄
> 2. 서버에서 SYN(클라이언트와 연결 여부)와 ACK(확인여부)를 보냄
> 3. 클라이언트에서 ACK(연결여부)를 서버에 보냄

-> 3way handshake 연결은 <font color="#ffff00">가상연결!</font>

## 순서 보장

![[Pasted image 20241231123526.png]]
- 순서를 파악하고, 순서대로 다시 보내게 함(다른 로직일 수도 있음 - 최적화)
