# 4. HTML Form 데이터 전송

#### 4.1 POST
![[Pasted image 20250102143948.png]]
- HTML form에 action="/save" method="<font color="#ffff00">post</font>" 이렇게 명시를 하고
	- input type = "text", name = "username"
	- input type = "text", name = "age"
- Body에서 거의 쿼리 파라미터와 비슷한 방식으로 값이 전송

#### 4.2 GET
![[Pasted image 20250102144203.png]]
- GET으로도 보낼 수 있다. -> GET으로 바꾸고 쿼리 파라미터로 전송
	- 근데 위의 예시에서 /save인데 get으로 쓰면 안된다.(get은 조회만)

#### 4.3 멀티파트/폼-데이터
![[Pasted image 20250102144332.png]]
- text뿐만 아니라, <font color="#ffff00">바이트 파일까지 전송</font>
	- Content-Type : multipart/form-data
- 각각의 데이터를 "-------"으로 경계를 나누어서 전송
	- 각각의 데이터에 대한 Content-Type을 명시


![[Pasted image 20250102144722.png]]
- 기본적으로 <font color="#ffff00">POST</font> 전송
- <font color="#ffff00">Content-Type: application/x-www-form-urlencoded 사용</font>
	-  메세지 바디를 통해서(key-value 쿼리 파라미터 방식으로 전송)
	- 전송 데이터를 url encoding처리
- GET으로도 가능함
- 멀티파트/폼-데이터 -> 이미지 파일과 같은 바이트 파일을 같이 보냄
	- "-----"으로 나누어서 같이 넘김
- HTML Form 전송은 GET, POST만 지원함
