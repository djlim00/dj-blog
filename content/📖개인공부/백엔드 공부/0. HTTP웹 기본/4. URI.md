>
> <font color="#ffff00">0. shcenme</font> : 주로 프로토콜
>    • 프로토콜 : 어떤 방식으로 자원에 접근할 것인가 결정한 규칙
>    • http, https
> 
> 1. **userinfo** : URL에 사용자 정보를 포함해서 인증
> 	• 거의 사용하지 않음
> 
> **<font color="#00b0f0">2. host</font>** : www.google.com 처럼 호스트명 -> 도메임명 또는 ip주소 직접 사용
> 
> <font color="#ff0000">3. port</font> : PORT주소를 나타냄 -> 일반적으로 생략(위의 내용)
> 	 • <font color="#ffff00">http = 80 port, https = 443 port (생략가능)</font>
> 
> <font color="#de7802">4. path</font> : 리소스 경로 -> 디렉토리 처럼 계층적 구조
> 	• /home/file1.jpg, /members, /members/100, /items/iphone12
> 	  
> <font color="#92d050">5. query</font> : key = value형태, '?'로 시작, '&'로 추가가능
> 	•  ?keyA=valueA&keyB=valueB
> 	•  query parameter, query string 등으로 불림.
> 	•  웹서버에서 제공하는 파라미터, 문자 형태
> 
> 6. fragment : html 내부 북마크
