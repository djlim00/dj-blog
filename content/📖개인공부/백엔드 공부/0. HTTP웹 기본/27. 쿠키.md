# 3. 쿠키의 사용
![[Pasted image 20250103013638.png]]
- 주로 <font color="#ffff00">사용자 로그인</font>에 쓰임
- session key를 만들어서 값을 클라이언트에 반환
	- 서버에 요청할 때 마다 sessionId를 전송
- 광고 정보 트래킹
- <font color="#ffff00">쿠키 정보는 항상 서버에 전송</font>
	- 네트워크 추가 트래픽 발생 -> 최소한의 정보만 사용하자
		- 인증 토큰, 아이디 정보 정도
- 서버에 전송하지 않고, 웹 브라우저 내부에 데이터를 저장
	- 웹 스토리지(localStorage, sessionStorage)

- 주의!! : 보안에 민감한 데이터는 저장 X (주민번호, 전화번호, 카드 번호)\

# 4. 쿠키의 생명 주기

![[Pasted image 20250103014012.png]]


>- **expires** : 만료 시점
>- **max-age** : 유지 시간(0이나 음수 이면 쿠키 삭제)

- 세션 쿠키 : 만료 날짜를 생략하면 브라우저 종료 시 까지만 유지
- 영속 쿠키 : 만료 날짜를 입력하면 해당 날짜까지 유지

# 5. 쿠키 - 도메인

- 아무 사이트에서나 쿠키를 전송하면 안됨!!

![[Pasted image 20250103014250.png]]
- <u>명시한 문서 기준 도메인 + 서브 도메인</u>
- 생략 -> 현재 딱 해당 도메인에서만 쿠키 생성, 하위는 생성 안함
