# 1. Cache-Control (캐시 제어 헤더)

1. **Cache-Control: max-age** : <font color="#ffff00">캐시 유효 시간</font>(초 단위)
2. **Cache-Control: no-cache** : 데이터는 캐시해도 되지만, <font color="#ffff00">항상 원(origin) 서버에 검증</font>하고 사용
3. **Cache-Control: no-store** : 데이터에 민감한 정보가 있으므로 <font color="#ffff00">저장하면 안됨</font>
(메모리에서 사용하고 최대한 빨리 삭제)
