
본격적으로 컨트롤러와 타임리프 뷰 템플릿으로 동적인 html을 만들어보자!

![[Pasted image 20250311233837.png]]
- 기존의 static 아래에 있는 html파일들은 타임리프를 쓸 수 없음!
- 타임리프를 쓰려면 template 아래에 만들어야함 -> 지금은 html파일들도 타임리프용이 아니라서 수정을 해야함~


![[Pasted image 20250312085852.png]]
- 이렇게 link태그에 th:href"@{.../.../..}" 를 넣어주면 타임리프에서 css를 가져오게 되서 css가 적용됨!
	- 둘다 해놓고 둘 중에 하나를 적용하는 것임


```java title:"타임리프가 적용된 template/basic/item.html",, hl:5,18,27-30
<!DOCTYPE HTML>  
<html xmlns:th="http://www.thymleaf.org">  
<head>  
    <meta charset="utf-8">  
    <link th:href="@{/css/bootstrap.min.css}"  
            href="../css/bootstrap.min.css" rel="stylesheet">  
</head>  
<body>  
  
<div class="container" style="max-width: 600px">  
    <div class="py-5 text-center">  
        <h2>상품 목록</h2>  
    </div>  
    <div class="row">  
        <div class="col">  
            <button class="btn btn-primary float-end"  
                    onclick="location.href='addForm.html'"  
                    th:onclick="|location.href='@{/basic/items/add}'|"  
                    type="button">상품 등록</button>  
        </div>    </div>  
    <hr class="my-4">  
    <div>        <table class="table">  
            <thead>            <tr>                <th>ID</th>  
                <th>상품명</th>  
                <th>가격</th>  
                <th>수량</th>  
            </tr>            </thead>            <tbody>            <tr th:each="item : ${items}">  
                <td><a href="item.html" th:href="@{/basic/items/{itemId}  
(itemId=${item.id})}" th:text="${item.id}">회원id</a></td>  
                <td><a href="item.html" th:href="@{|/basic/items/${item.id}|}" th:text="${item.itemName}">상품명</a></td>  
                <td th:text="${item.price}">10000</td>  
                <td th:text="${item.quantity}">10</td>  
            </tr>            </tbody>        </table>    </div>  
</div> <!-- /container -->  
  
</body>  
</html>
```
- 이런식으로 th를 붙이는 타임리프 문법을 사용하면, 리다이렉트를 할 수 있다
- 또한 반복문으로 이렇게 Item값을 가져와서 목록을 띄우는 기능도 가능하다 -> 깨지지않음!
	- <mark style="background: #FFF3A3A6;">타임리프로 작성된 코드는 서버가 띄워져 있지 않아도, 그냥 html로 인식(th부분을 무시)하기 때문에 정상적으로 동작한다 -> Natural Template이라고 부르는 이유!</mark>

![[Pasted image 20250312092943.png]]
- th: href를 하면 뷰 템플릿을 거치면 가는 링크, 그냥 href는 일반적으로 html(대체 됨)
	- 내츄럴하다 내츄럴 해..
- 대부분의 HTML 속성을 th : xxx 이런식으로 바꿀 수 있다.
- URL 링크를 사용하는 경우 @{ ... } 이런식으로 표현한다.
