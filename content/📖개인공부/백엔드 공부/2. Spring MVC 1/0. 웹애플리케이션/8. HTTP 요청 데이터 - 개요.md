
- <font color="#ffff00">"GET" - 쿼리 파라미터</font>
	- /url**?username=hello&age=20**
	- 메시지 Body없이, URL의 쿼리 파라미터에 데이터를 포함해서 전달
	- ex) 검색, 필터, 페이징 등에서 많이 사용

- <font color="#ffff00">"POST"</font>
	- <font color="#ffff00">HTML Form 방식</font>
		- content-type: application/x-www-form-urlencoded
		- 메시지 바디에 쿼리 파라미터 형식으로 전달 -> username=hello&age=20
		- ex) 회원 가입, 상품 주문, HTML Form 사용

- <font color="#ffff00">"POST", "PUT", "PATCH"</font>
	- <font color="#ffff00">HTTP Body에 데이터 직접 담아서 요청</font>
		- HTTP API에서 주로 사용
		- JSON, XML, TEXT
		- 주로 <u>JSON을 사용</u>
