___
## 2.1 서버 사이드 랜더링 -> 백엔드

![[Pasted image 20250105092812.png]]
- 웹 브라우저에서 주문 내역 요청 -> 서버에서 DB를 조회 -> HTML 생성 -> 서버에서 모두 다 랜더링 완료 -> 응답으로 보내줌 -> 클라이언트는 그냥 받아서 보여주기만 함
- <font color="#ffff00">동적으로 생성된 HTML이 전부 서버에서 생성(랜더링)된다</font>
	- JSP, 타임리프

___
## 2.2 클라이언트 사이드 랜더링 -> 웹 프론트엔드 

![[Pasted image 20250105093030.png]]

- 자바 스크립트를 사용해서 HTML을 동적으로 생성한다 -> 웹 브라우저의 자바 스크립트를 이용해서 만듦
- <font color="#ffff00">동적으로 HTML이 클라이언트에서 생성(랜더링)된다.</font>
	- React, Vue.js
