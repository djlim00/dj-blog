# 2. 자바 뷰 템플릿 역사
#### HTML을 편리하게 생성하는 뷰 기능

- JSP
	- 속도 느림, 기능 부족
	- Spring에서 권장하지 않음
- 프리마커(Freemarker), Velocity(벨로시티)
	- 속도 문제 해결, 다양한 기능
- <span style="background:#fff88f">타임리프(Thymeleaf)</span>
	- 내추럴 템플릿 : HTML의 모양을 유지하면서 뷰 템플릿 적용 가능
	- 스프링 MVC와 강력한 기능 통합
	- **최선의 선택**, 단 성능은 프리마커, 벨로시티가 더 빠름
