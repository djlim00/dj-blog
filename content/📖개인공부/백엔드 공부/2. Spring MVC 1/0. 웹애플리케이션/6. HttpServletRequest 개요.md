
**"HttpServletRequest"의 역할**
: 개발자가 모든 HTTP 요청 메세지를 파싱해서 받아들이고, 텍스트를 작성해서 응답을 해도 되지만, 굉장히 귀찮은 일이다. 그걸 이제 HttpServletRequest이 대신하여서 편리하게 조회할 수 있게 해준다.

"HttpServletRequest의 부가기능"

1. 임시저장소 기능
- 해당 HTTP 요청이 시작부터 끝날 때 까지 유지되는 임시 저장소 기능
	- 저장 : request.setAttribute(name, value)
	- 조회 : request.getAttribute(name)

2. 세션 관리 기능
- request.getSession(create: true)
