- <font color="#d83931">우리는 이제 컨트롤러에서 메서드 단위로 @RequestMapping을 적용하기 때문에 만들기 때문에 하나의 @Controller를 이용해서 통합을 할 수 있다.</font>

```java title:"SpringMemberController - V2"
@Controller  
public class SpringMemberControllerV2 {  
  
    private MemberRepository memberRepository = MemberRepository.getInstance();  
  
    @RequestMapping("/springmvc/v2/members/new-form")  
    public ModelAndView newForm(){  
        return new ModelAndView("new-form");  
    }  
  
    @RequestMapping("/springmvc/v2/members")  
    public ModelAndView save() {  
        List<Member> members = memberRepository.finfAll();  
        ModelAndView mv = new ModelAndView("members");  
        //mv.getModel().put("members",members);  
        mv.addObject("members",members);  
  
        return mv;  
    }  
  
    @RequestMapping("/springmvc/v2/members/save")  
    public ModelAndView members(HttpServletRequest request, HttpServletResponse response) {  
        String username = request.getParameter("username");  
        int age = Integer.parseInt(request.getParameter("age"));  
  
        Member member = new Member(username, age);  
        memberRepository.save(member);  
  
        ModelAndView mv = new ModelAndView("save-result");  
        mv.getModel().put("member",member);  
        mv.addObject("member",member);  
        return mv;  
    }  
}
```
- 이렇게 메서드 레벨로 모아둘 수 있다!
