요청 매핑 - @RequestMapping

요청데이터 - 쿼리파라미터, html폼, http body
	요청 파라미터 : @RequestParam, @ModelAttribute
	 http 바디 : 보통 JSON - ResponseEntitiy, @RequestBody
		 JSON도 @RequestBody로 처리하면 됨

응답데이터 - 정적, 뷰 템플릿, Http body
	 바디 : ResponseEntitiy<>(), @ResponseBody
	 @RestController = @ResponseBody + @Controller
