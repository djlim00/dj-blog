- 실제로는 요청 파라미터를 가지고 객체를 만들고, 그 객체에 값을 넣어줘야함.
	- 이걸 그냥 @ModelAttribute를 쓰면 한번에 만들어진다.


```java title:"HelloData 클래스"
package hello.springmvc.basic;  
  
import lombok.Data;  
  
@Data  
public class HelloData {  
    private String username;  
    private int  age;  
}
```
![[Pasted image 20250311091411.png]]

```java title:"기존의 방식"
@ResponseBody  
@RequestMapping("/model-attribute-v1")  
public String modelAttribute(@RequestParam String username, @RequestParam int age){  
    HelloData helloData = new HelloData();  
    helloData.setUsername(username);  
    helloData.setAge(age);  
    return "ok";  
}
```
- 기존에는 @RequestParam을 가지고 하나씩 가져와서 만든다.


```java title:"@ModelAttribute"
@ResponseBody  
@RequestMapping("/model-attribute-v2")  
public String modelAttribute(@ModelAttribute HelloData helloData){  
  
    log.info("helloData:{}" , helloData);  
    return "ok";  
}

@ResponseBody  
@RequestMapping("/model-attribute-v2")  
public String modelAttribute(HelloData helloData){  
  
    log.info("helloData:{}" , helloData);  
    return "ok";  
}
```
- 이거 진짜 말 안되네..
