#### 🔥 >>연산은 여전히 시프트연산이 맞다, range(x, y)는 x에서 y-1까지
```python
a = 100
result = 0
for i in range(1,3):
   result = a >> i
   result = result + 1
print(result)
```
- 가만히 보면 a의 값은 바뀌지 않는다. 
- 따라서 100 >> 2^2 +1 한 값이 그냥 정답임. 그러면 100/4 +1 = 25+1 = 26
답 : 26
