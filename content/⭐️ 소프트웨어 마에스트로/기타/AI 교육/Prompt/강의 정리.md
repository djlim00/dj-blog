# Agentic AI

![[Pasted image 20260501131702.png]]
#### 1. 자기성찰(Reflection) : AI가 스스로 생성한 결과물을 검토하고, 수정하여 품질을 높이는 과정
#### 2. 도구 사용(Tool Use) : AI가 외부 API나 도구(웹 검색, 코드 실행 등)를 활용하는 능력
#### 3. 계획 수립(Planning) : 복잡한 목표를 하위 작업으로 분해하고 실행 순서를 결정하는 것
#### 4. 다중 에이전트 협업(Multiagent Collaboration) : 서로 다른 역할(개발자, 디자이너 등)을 부여받은 여러 AI가 대화하며 협력하는 방식
