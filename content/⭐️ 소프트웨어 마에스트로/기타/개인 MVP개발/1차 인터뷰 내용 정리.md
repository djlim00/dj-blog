
# 정리1
- 법인세 신고시에 회계상이익과 세무상이익이 다른데 싱크맞출때 세법개정시에 기존 계산과 달라질 수 있음 세액공제나 이럴때 체크를 해주고 계산해주면 좋다
- 자료를 연도별로 회사별로 업무별로 정리를 하는데 나중에 좀 뒤섞인다 ai가 정리를 해주면 좋을 수 있음
- 세무는 소득세, 상속세 등등 분류하는데 법인세는 회사별로 정리하는게 일이 잡무를 줄이기 가능해보임
- 회사에 대해 회계감사중인데 용역했던것들이 필요할때 그 파일 찾아주고하면 좋다
- ★엑셀과 한글 표나 그래프같은게 호환이 잘 되지않아서 갖다 붙여넣기 할때 불편할 수 있음
- 더존,위하고를 쓰는 회사는 호환되는 자료받기가 좋은데 안쓰면 엑셀로 받아야함
- 더존,위하고 프로그램은 해당 회사의 장부 볼 때 쓴다
- 큰회계법인에서는 장부 작성일은 안하고 세무사 사무실이나 작은 회계사무실에서 한다
- 일반인들이 질문해서 ai로 틀린 정보들을 가져오는 경우가 꽤 있다

1. 어떤 자료가 오나요?(이메일, 문서..etc) 어떤 형태
    1. _회사 손익, 매출, 재무재표 등등의 자료들_
    2. _엑셀파일을 많이 사용중_
    3. _조서에 히스토리를 많이 남김_
    4. _히스토리란 회사의 업종이나 현황, 주주분포, 차입이 줄었다던가 추세등을 파악할때 필요 이런 뭔가 특이사항등을 남김_

2. 현재 주로 사용하시는 프로그램이 뭐예요? (더존 스마트A, 위하고T, 세무사랑 등)
    1. _회계감사, 컨설팅(주당가격평가), 조서, 보고서 이런거 작성 할땐 사용하지 않음_
    2. _office 프로그램을 주로 사용하심_
    3. _큰 기업에서는 ai 툴이나 프로그램 사용이 꽤 잘 되어서 신입들의 일들을 파이를 많이 뺐긴상태_
    4. _하지만 개인 사무실에서는 아직 툴에 대한 사용이 상용화 되어있지 않은듯함_
    5. _작업 시간은 년수만 넣으면 바로 나오는것도 있으나 오래걸리는 경우도 있다_
    6. _기존하던 업무 아닌 이상 쉽지는 않음_
    7. _감사는 매년 진행됨 - 메모는 워드 엑셀 등

# 정리2

- 회사의 회계 프로그램의 파일들을 다운로드 -> 아예 전부 돌려서(이상거래 탐지)

- 조서 -> 보고서(워드, ppt)

- 주식 가치 평가
    - 가치 평가(1000원이 나오는 과정 -> 엑셀) / A기업의 방식을 B기업으로 바꾼다.
    - 해본 사람한테 전화해서 (새로운 방식을 했던 사람한테 연락을 해서 파일(보고서, 했던 방식을)을 받는다)
        - 아는 사람 없으면 자기가 열심히 공부해야한다.

- 팀별로 하게 되면 -> 서로 토의하면서 해당 케이스에 대해서 알아보기
    - 큰 회사들은 자료구하기 쉬울 것

- 회계감사(회계 자료를 받아야함)
    1. 재무재표를 제대로 됐는지를 확인해야하니까(회계 자료는 더존, 위하고, 혹은 자체적으로)
        - 저걸 프로그램을 받아서, 이제 엑셀로 export
- 가치평가
    1. 어떤 재산이 있는지, 손익현황, 재무상태표를 받아야함.
    2. 회사 내부 사정
    3. +판단

 - **법인세 신고(세무조정)**
    - 회계상 이익 vs 세무상 이익
    - 회계상 이익 -> 세무상 이익 -> 거기에 20%
    - 세법이 개정이 되서 적용이 되거나/안되거나
    - 기한이 있는 혜택 같은 것들은 -> 반영이 안될 수도 있음. 알려주면 좋음!
    - 이 회사의 주요 이슈 사항

- **파악하는 히스토리**
    - 조서, 메모, 워드, 엑셀
    - 보고서에는 안들어감
    - 보는 이유 : 회사 현황(업계 현황, 회사의 주주 분포, 차입이 줄었다, 추세)
        - 그냥 "~하는 중임"
        - 영구조서 : 매년 주요 현황을 기록하고 관리하고 있다.


- **사람마다 폴더 정리하는 방식이 다름**
    - 년도/회사/업무(회계감사-년도 / 세무-법인세,소득세,상속세 / 용역-컨설팅)
    - 어떤건 년도 별로 하는게 좋고, 어떤건 회사별로 하는게 좋은데,,, -> 이게 오래되면 막 섞임
        
    - 안에 내용까지 연관성을 가진 파일을

    - 한글파일(hwp)이 문제다.
