# LLM 트랜드 및 유플러스의 전략

1. 
- Proprietary Model과 Open Model의 격차가 감소되며 <span style="background:#fff88f">상향 평준화</span>
- Pretraining의 중요도 대비, Post-training의 중요도가 증대
	- Scaling law의 한계에 봉착함 -> Post-training의 중요성 증대

2. 
- Llama의 부진과 중국발 LLM(ex. Qwen)의 약진

3. 
- 소버린 AI 시대의 개막 -> 독자 AI 파운대이션 모델 프로젝트 착수
- 전형적인 경우는 Cloud LLM이 좋긴한데
	- Private 데이터를 다룰 경우는 Cloud API 형태의 LLM은 리스크가 존재
		- 기업 내부 정보
		- 고객 개인 정보

5. 
- 대부분은 LLM을 위한 인프라가 부족하므로, sLLM을 이용해야함
	- 그렇기 때문에 sLLM을 이용하기 위해서 굉장히 좁은 분야를 잘 잡아서 아용해야함(뾰족한 도메인)
- 공개 LLM은 private 데이터가 없음

6. 
- DACP는 Fine-tuning과 차이 -> DACP는 개념 설명, Fine-tuning은 문제 풀이 형식(굉장히 큰 학습 데이터가 구성되어서 질문0답변 형태의 데이터 셋으로 학습)

7.

	- 성능 같은 것 확인할 수 있음.
