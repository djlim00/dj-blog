: <font color="#ffff00">첫문자 대문자 + 카멜 케이스 + 숫자는 맨앞 말고는 가능(특수문자: ' $ ', ' _ ' 가능 but 비추)</font>
	- ' $ ' : inner class의 경우에 outer class에 대해서 A.class$B.class 이렇게 되기 때문에 $가 있으면 좋지 않음
	- ' _ ' : .jsp를 쓸대 { Test.jsp -> Test_jsp.java -> Test_jsp.class } 이렇게 변환되기 때문에 ' _ '을 사용하면 좋지 않음
		- JSP는 서블릿을 쉽게 코딩하기 위해서 만들었음 -> 쉽게 코딩하면 서블릿으로 변환되고 -> 변환된 코드가 이제 컴파일 되어서 .class 파일이 되는 것이다.
: 클래스명은 소스파일명과 같아야 한다.

# 타입 선언

## 1. 변수 선언

- 변수(variable)이란 하나의 값을 저장할 수 있는 메모리 공간에 붙여진 이름
- 자바는 변수의 타입이 굉장히 엄격하게 지켜진다()


- **Primitive Data type** : namespace에 선언된 변수 a에 주솟값이 아니라 그냥 값 자체를 가짐
ex) 
int a = 7;
int b = 8;
a == b인가? ->  참이다 

- **Objective Data type** : namespace에 선언된 변수 a에 주솟값을 가진 데이터가 있고, 해당 데이터의 주솟값에 진짜 데이터가 있다
ex)
String s = new String("hi");
String s2 = new String("hi");
s == s2인가? ->  참이 아니다! 주솟값이 다르기 때문이다!

== 이 아닌 equals를 사용하면 주솟값이 아닌 value값을 가져와서 비교하기 때문에 값을 비교할 수 있다.
	-> equals를 override를 하면 내용 비교를 할 수 있는 메서드를 만들 수 있다.

- double 계산이 float에 비해 더 정밀하고, 더 큰 실수를 저장할 수 있음.

### 2. 타입 변환

- 자동 타입변환 -> 작은 타입이 큰 타입으로 대입될 때 발생 ex) byte -> short
	- <font color="#ffff00">정수타입이 실수 타입으로 대입되면 무조건 자동 변환이 됨 </font>ex) int -> double
- 명시적 타입변환(강제 캐스팅)
	- 큰 타입 -> 작은 타입
	- 데이터의 손실(오버플로우)가 날 가능성을 가진다!

<span style="background:#fff88f">객체의 캐스팅이 중요함!</span>

은근히 char -> int 형변환은 자주 사용됨



___
```java title:"변수관련 캐스팅 코드 예시"
package ch02;  
  
/*  
 *   Data Type Casting *   
 *   - Promotion(묵시적/자동 형태 변환) : 작은 데이터 타입 -> 큰 데이터 타입  
 *   - Demotion(명시적/강제적 형태 변환) : 큰 데이터 타입 -> 작은 데이터 타입  
 */  
  
public class CastingTest {  
    public static void main(String[] args) {  
        byte bVa;  
        bVa = 2;  
        int iVa = bVa; // 자동 형변환  
  
        int iVa2 = 128; // -128 ~ 127가 byte의 범위임  
        byte b2Va = (byte) iVa2; // 명시적(강제적) 형변환  
  
        System.out.println("자동 형태변환 : " + iVa);  
        System.out.println("강제 형태변환 : " + b2Va); // 오버플로우 발생  
  
        int iVar3 = 999999999;  
        double dVar = iVar3;  
        System.out.println("lVar = " + dVar);  
  
        double dVar2 = 9.123456789;  
        int iVar4 = (int) dVar2;  
        System.out.println("iVar4 = " + iVar4); // 소수점이 날라  
  
        char chVar = 'Z';  
        System.out.println("chVar : " + chVar);  
        System.out.println("(byte) chVar : " + ((byte) chVar));  
  
        int iVar5 = chVar;  
        System.out.println("iVar5 = " + iVar5);  
  
        byte bVar6 = 7, bVar7 = 8;  
        int iVar7 = bVar6 + bVar7;  
        byte bVar8 = (byte) (bVar6 + bVar7); // 더하기 연산의 결과는 int이므로 형변환을 해야함  
        System.out.println("iVar7 = " + iVar7);  
  
        float flVar1 = 99.9F, flVar2 = 3.14f;  
        float flVar3 = flVar1 + flVar2;  
        System.out.println(flVar3);  
  
        long lVar = 999999L, lVar2 = 777777l;  
    }  
}
```
