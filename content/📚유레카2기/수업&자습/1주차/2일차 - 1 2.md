## 1.1 증감연산자

- ++, -- : 앞에 쓰면 먼저 수행하고 대입, 뒤에 쓰면 대입하고 증가.

```java title:"순서에 따른 변화", hl:8,11
package ch03;  
  
public class OperationTest {  
    public static void main(String[] args) {  
  
        int a = 7, b = 7, c = 0, d = 0;  
  
        // c에 a를 대입하고, a를 1증가  
        c = a++;  
  
        // b를 1증가 시킨 다음에, d에 대입  
        d = ++b;  
        
        System.out.println("c = " + c);  
        System.out.println("d = " + d);  

		a = 7;  
		b = 7;  
		c = 0;  
		d = 0;  

		// ++가 뒤에 붙었으므로 a는 3이 곱해지고 c에 대입된 후에 1증가
		c = a++ * 3;
		// ++가 앞에 붙었으므로 b는 1증가되고 3이 곱해짐  
		d = ++b * 3;  
		  
		System.out.println("c = " + c);  
		System.out.println("d = " + d);
    }  
}

====================================================================

c = 7
d = 8
c = 21
d = 24
```

## 1.2 오버플로우 & 언더플로우

- 어제 강제 캐스팅하는 과정에서 int -> byte로 바꾸게 되면 에러가 발생함
	- 기존의 큰 비트수로 나타내야하는 숫자가 작은 비트로 바뀌게 되면서 비트가 튀면서 발생하는 현상
	- 작은 -> 큰 으로 바꾸는 묵시적 캐스팅의 경우는 그냥 비트가 늘어나게 되는 것이니까 발생하지 않음


```java title:"byte타입의 범위를 넘는 값이 들어감"
byte bVar1 = (byte) 128;  
System.out.println("bVar1 = " + bVar1);
============================================================
bVar1 = -128
```

## 1.5 논리연산자 && , ||, ^, !

- &&(AND) : 둘다 true면 true
	-  & : 두개 모두 true인지 확인
	- && : 첫번쨰 검사에서 false면 바로  false
- ||(OR) : 둘 중 하나만 true여도 true
	- | : 연산자 하나만 쓴 경우에는 무조건 두 개를 전부 검사한다
	- || : 첫번째 검사에서 true가 나오면 바로 true를 반환
- <font color="#ffff00">^(XOR) : 둘 중 하나만 true 여야만 true</font>
- !(NOT) : 논리값 반전

```java title:"숏 서킷 로직 예시", hl:13-18
package ch03;  
  
public class ShortTest {  
    public static void main(String[] args) {  
  
        System.out.println(1 < 2);  
        
        if (1 < 2 && 0 < 2) {  
            System.out.println("TRUE");  
        }  
  
  
        if (1 < 2 || (5 / 0) < 2) {  
            System.out.println("TRUE");  
        }  
//        if (1 < 2 | (5 / 0) < 2) {  
//            System.out.println("TRUE");  
//        }  
  
        // 숏 서킷 로직  
        if (1 < 2 || (5 / 0) < 2) {  
            System.out.println("TRUE");  
        }  
    }  
}

==========================
true
TRUE
TRUE
TRUE
```

### 2.1 switch문

- 조건에는 boolean, 실수형 사용 못함
- <font color="#ffff00">조건에는 정수형, char, String과 같은 문자와 문자열 가능함</font>

```java title:"switch문 예시"

String grade = "A";  
switch (grade) {  
    case "A":  
        System.out.println("A");  
        break;  
    case "B":  
        System.out.println("B");  
        break;  
    case "C":  
        System.out.println("C");  
        break;  
    default:  
        System.out.println("F");  
        break;  
}

```

- <font color="#ffff00">break를 하지 않으면 해당 케이스 아래의 모든 case를 실행하면서 내려온다.</font>

## 2.2 break & continue

- break : <u>가장 가까운</u> 반복문을 탈출한다

```java title:"예시"
package ch03;  
  
public class BreakTest {  
    public static void main(String[] args) {  
  
        for (int i = 1; i <= 10; i++) {  
            if (i != 8) System.out.println("i = " + i);  
            else break;  
        }  
  
        System.out.println("_____________________________");  
        
        for (int i = 1; i <= 10; i++) {  
            if (i == 8) continue;  
            else System.out.println("i = " + i);  
  
        }  
    }  
}
===================<결과>====================
i = 1
i = 2
i = 3
i = 4
i = 5
i = 6
i = 7
_____________________________
i = 1
i = 2
i = 3
i = 4
i = 5
i = 6
i = 7
i = 9
i = 10
```
