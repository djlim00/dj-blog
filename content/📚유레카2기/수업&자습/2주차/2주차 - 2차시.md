# 네트워크의 시작

- 냉전시절 미국의 아르파넷 -> 위싱턴에서 핵 미사일 기지인 텍사스?까지 신호를 보내기 위해 만들었음
	- 거리가 너무 멀고 수 많은 노드가 있기 때문에, 정확하게 신호를 보내기 위해서는 
		- **IP주소와 port**를 이용한다.
	- 네트워크에서 목적지를 찾아가지 못하고, 죽은 데이터가 네트워크에 남아있게 된다
		- **TCP** : IP 주소, 포트를 이용해서 전송제어의 개념이 추가
		- 알맞은 목적지까지 도달할 수 있게 해준다.
		- ttl이 도입되어서 유통기한이 지나면 폐기됨


### TCP vs UDP

- UDP는 전송을 하면 끝이다 (별도의 확인 작업이 없음)
- TCP는 syn -> ack+syn -> ack (3-way handshake) 가 지나야지 전송을 한다.

##### "3-way handshake"
![[Pasted image 20250204100757.png|500]]
Server에서 열려있는 포트는 LISTEN 상태이고 Client에서는 Closed 상태이다.

1. Client에서 Server에 연결 요청을 하기위해 SYN 데이터를 보낸다.
   
2. Server에서 해당 포트는 LISTEN 상태에서 SYN 데이터를 받고 SYN_RCV로 상태가 변경된다.
   그리고 요청을 정상적으로 받았다는 대답(ACK)와 Client도 포트를 열어달라는 SYN 을 같이 보낸다.
   
3. Client에서는 SYN+ACK 를 받고 ESTABLISHED로 상태를 변경하고 서버에 요청을 잘 받았다는 ACK 를 전송한다.
   
4. ACK를 받은 서버는 상태가 ESTABLSHED로 변경된다.

위와 같이 3번의 통신이 정상적으로 이루어지면, 서로의 포트가 ESTABLISHED 되면서 연결이 되게 된다.

> 서버는 한개인 반면 클라이언트는 N개 이므로, 서버에서는 클라이언트의 연결 요청을 대기하는 포트 하나와 나머지 연결된 포트, 연결이 아직 안된 포트를 관리한다.

### 즉 TCP는"연결지향"

- 전화는 통신자와 수신자가 확실히 연결이 되었는지 확인을 하고 연결을 함 - 가상연결
- TCP의 연결은 가상연결 -> 수신자의 IP+Port가 송신자의 IP+Port가 양 끝단이 열란 상태로 통신을 한다는 뜻이고, 그 사이의 네트워크가 직접적으로 연결이 되어있지는 않음.
	- HTTP는 심지어 비연결지향임ㄷㄷ(Session, Cookie 사용합니다).
- 소실되는 패킷이 있다면 ACK+SYN이 오지 않고 일정시간 기다리다가 다시 보내줌

# 간단한 서버 클라이어트 통신 코드

```java title:"서버 코드"
package ch19Network;  
  
import java.io.DataOutputStream;  
import java.io.IOException;  
import java.io.OutputStream;  
import java.net.ServerSocket;  
import java.net.Socket;  
  
public class Server1Simple {  
    public static void main(String[] args) throws IOException {  
        // 포트번호 54321번을 연다.  
        ServerSocket serverSocket = new ServerSocket(54321);  
        // socket은 아직 null        // OutputStream으로 받아오  
        // 바이트로 읽는 것은 힘드므로 DataOutputStream으로 한번 필터림  
        Socket socket = null;  
        OutputStream os = null;  
        DataOutputStream dos = null;  
        System.out.println("Server Started................");  
  
        while (true) {  
            socket = serverSocket.accept();  
            System.out.println("Server Accept...............");  
  
            os = socket.getOutputStream();  
            dos = new DataOutputStream(os);  
  
            dos.writeUTF("Server accept client request.................");  
  
            dos.close();  
            os.close();  
        }  
    }  
}
```

```java title:"클라이언트 코드"
package ch19Network;  
  
import java.io.DataInputStream;  
import java.io.IOException;  
import java.io.InputStream;  
import java.net.Socket;  
  
public class Client1Simple {  
    public static void main(String[] args) throws IOException {  
  
        Socket socket = new Socket("127.0.0.1", 54321);  
        InputStream is = socket.getInputStream();  
        DataInputStream dis = new DataInputStream(is);  
  
        String serverMessage = dis.readUTF();  
        System.out.println(serverMessage);  
  
        dis.close();  
        is.close();  
    }  
}
```
