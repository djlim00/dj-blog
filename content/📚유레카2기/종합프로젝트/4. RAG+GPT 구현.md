# 구현 설명

## 폴더링
![[Pasted image 20250601213935.png]]
- **/app** : 실질적인 기능을 하는 코드.
	- **build_index.py** : 요금제 데이터를 벡터화하여 FAISS 인덱스를 생성합니다.
	- **main.py** : FastAPI 서버를 실행하고, HTTP 요청을 처리합니다.
	- **rag_chain.py** : LangChain으로 구성된 RAG 질의응답 체인을 정의합니다.
	- **utils.py** : 금칙어 필터링 등 부가 기능을 위한 유틸리티 함수들이 들어있습니다.
- **/data** : 벡터DB에 들어가는 요금제 데이터 txt파일.
- **/faiss_index** : FAISS 벡터 인덱스 관련 파일.
	- **index.faiss** : FAISS 라이브러리로 생성된 **벡터 인덱스 파일**로, 텍스트 임베딩 벡터들의 실제 검색용 인덱스가 저장된다. 검색 속도와 효율성을 높이기 위한 핵심 구조.
	-  **index.pkl** : 벡터 인덱스와 매핑되는 **문서 ID → 문서 메타 정보**(예: Document 객체 등)를 저장한 파일. LangChain의 `FAISS.save_local()` 함수로 자동 생성.
	- **text.pkl** : 벡터로 변환된 원본 텍스트 리스트를 저장한 파일. 사용자가 입력한 질의(query)와 유사한 문서를 다시 보여주기 위해 사용.
- **.env**  : API_KEY 환경변수 파일.
- **requirements.txt** : 필요한 라이브러리 목록(pip로 한번에 설치하기 위해).


## 1. build_index.py
```python 
# build_faiss_from_plans.py  
import os, pickle, faiss  
from langchain_community.embeddings import HuggingFaceEmbeddings  
from langchain_community.vectorstores import FAISS  
from langchain_core.documents import Document  
  
# 1. plans.txt 로드  
with open("data/plans.txt", "r", encoding="utf-8") as f:  
    raw_text = f.read()  
  
# 2. 문서 분리 (### 기준으로 나누기)  
chunks = raw_text.split("##############################")  
texts = [chunk.strip() for chunk in chunks if chunk.strip()]  
  
# 3. Document로 변환  
docs = [Document(page_content=t) for t in texts]  
  
# 4. 임베딩 모델 로딩  
embedding_model = HuggingFaceEmbeddings(model_name="upskyy/bge-m3-korean")  
  
# 5. 벡터스토어 생성 및 저장  
vectorstore = FAISS.from_documents(docs, embedding_model)  
vectorstore.save_local("faiss_index")  
  
# 6. 텍스트도 같이 저장  
with open("faiss_index/texts.pkl", "wb") as f:  
    pickle.dump(texts, f)  
```
- 챗봇의 답변을 위해서 존재하는 파일은 아님. 
- **plans.txt를 읽고 임베딩한 뒤, FAISS 벡터 인덱스를 생성하려 저장하는 인덱스 구축 스크립트**


## 2. main.py
```python
from fastapi import FastAPI, Request  
from pydantic import BaseModel  
from app.rag_chain import query_chain  
  
app = FastAPI()  
  
class ChatRequest(BaseModel):  
    question: str  
    chat_history: list[str] = []  
  
@app.post("/chat")  
def chat_endpoint(req: ChatRequest):  
    response = query_chain(req.question, req.chat_history)  
    return {"response": response}
```
- FAST API의 진입점 - "/chat" 엔드 포인트를 정의
- 사용자의 질문을 JSON의 "question" 파라미터로 받아서 `query_chain()` 함수를 통해서 응답 반환


## 3. rag_chain.py ⭐️
```python
import pickle, faiss  
import os  
from dotenv import load_dotenv  
from langchain_community.vectorstores import FAISS  
from langchain_community.embeddings import HuggingFaceEmbeddings  
from langchain_community.chat_models import ChatOpenAI  
from langchain.chains import ConversationalRetrievalChain  
from langchain.memory import ConversationBufferMemory  
from langchain.prompts import ChatPromptTemplate  
from langchain_core.documents import Document  
from app.utils import filter_profanity  

# 0. OPENAI_API_KEY 로드
load_dotenv()  
  
# 1. 텍스트 및 FAISS index 로드  
with open("faiss_index/texts.pkl", "rb") as f:  
    texts = pickle.load(f)  
index = faiss.read_index("faiss_index/index.faiss")  
  
# 2. 문서로 변환  
docs = [Document(page_content=t) for t in texts]  
  
# 3. 임베딩 모델 및 벡터스토어 구성  
embedding_model = HuggingFaceEmbeddings(model_name="upskyy/bge-m3-korean")  
vectorstore = FAISS.from_documents(docs, embedding_model)  
  
# 4. Retriever & Memory  
retriever = vectorstore.as_retriever(search_kwargs={"k": 3})  
memory = ConversationBufferMemory(memory_key="chat_history", return_messages=True)  
  
# 5. 프롬프트  
prompt = ChatPromptTemplate.from_template("""  
당신은 LG U+ 요금제 상담 챗봇입니다.  
고객의 질문에 정중하면서도 간결하게, 핵심 정보만 명확히 전달하세요.  
  
- 반드시 존댓말을 사용하세요.  
- 요금제 이름, 가격, 혜택을 항목별로 나눠서 요약하세요.  
- 맥락(context)에 포함된 요금제 중 가장 관련 있는 것을 추천하세요.  
  
고객 질문: {question}  
참고 요금제 정보: {context}  
  
답변:""")  
  
# 6. QA 체인  
qa_chain = ConversationalRetrievalChain.from_llm(  
    llm=ChatOpenAI(  
        model_name="gpt-4-1106-preview",  
        temperature=0.2,  
        openai_api_key=os.environ["OPENAI_API_KEY"]  
    ),  
    retriever=retriever,  
    memory=memory,  
    combine_docs_chain_kwargs={"prompt": prompt}  
)  
  
def query_chain(question, chat_history=[]):  
    # 금칙어 필터링 적용  
    clean_question = filter_profanity(question)  
  
    # 1. 벡터스토어에서 유사 문서 검색  
    docs = retriever.get_relevant_documents(clean_question)  
  
    print("\n[🔎 RAG - 벡터DB에서 검색된 문서 후보들]")  
    for i, doc in enumerate(docs, 1):  
        print(f"\n--- 문서 {i} ---\n{doc.page_content}")  
  
    # 2. QA 체인 실행 (ChatGPT가 답변 생성)  
    result = qa_chain.run({"question": clean_question, "chat_history": chat_history})  
    return result
```
